# 🎓 Knowledge Hub AI

### AI-Powered Learning Assistant for School Students

Knowledge Hub AI is an educational web platform designed to help students learn more effectively through Artificial Intelligence. The platform provides AI-generated study notes, topic-specific quizzes, and instant doubt-solving assistance while supporting **SDG 4: Quality Education**.

---

# 🚀 Features

## 📝 AI Notes Generator

Generate structured study notes from any academic topic.

### Capabilities

* Topic summaries
* Key concepts
* Important terminology
* Quick revision points
* Exam preparation support

---

## ❓ AI Quiz Generator

Generate topic-specific multiple-choice questions.

### Capabilities

* Subject-aware question generation
* MCQ format
* Four options per question
* Correct answers
* Detailed explanations
* Automatic score calculation

---

## 🤖 AI Doubt Solver

Interactive chatbot for academic support.

### Capabilities

* Real-time doubt solving
* Subject-specific responses
* Step-by-step explanations
* Student-friendly language

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/knowledge-hub-ai.git
```

## Navigate to Project

```bash
cd knowledge-hub-ai
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
```

## Start Development Server

```bash
npm start
```
